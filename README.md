# Nexora Digital — World-Class Company Profile Landing Page

Production-ready corporate premium landing page for a professional technology company / digital agency.

## Delivered
- Next.js 14 App Router + TypeScript
- Tailwind CSS premium deep-blue / white / gold design system
- Framer Motion animations
- shadcn/ui-style primitives
- React Hook Form + Zod validation
- Nodemailer-ready contact route
- EN/ID i18n dictionaries and locale routes
- Sanity CMS client + blog schema setup
- SEO metadata, sitemap.xml, robots.txt
- GA4 + Hotjar analytics hooks
- All requested sections from navbar through mega-footer

## Run
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Verify
```bash
npm run lint
npm run typecheck
npm run build
npm audit --audit-level=moderate
```
